Página Web responsiva para consumo de uma API de [pokémon](https://pokeapi.co/).

## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/ApenasoVini/API-Pokemon.git
```

Instale as dependências

```bash
  npm install
```

Abra a página

```bash
  npm start
```
